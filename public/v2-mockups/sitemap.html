Hierarchy Decisions — All 6 Resolved
✅Matches vs Tournaments — merged on one page
Open games and club tournaments live on a single chronological feed. No nav-level split.
Standard club tournaments (single day) → Matches page alongside open games
External/multi-day events (PPS, TPS, WPT) → dedicated detail pages reachable from the feed
✅Tournament results — live on tournament detail page
Results live on the tournament detail page itself (same URL, different state post-completion). No separate results section needed.
Homepage modules surface recent results (last 5–8) as clickable cards
Player profile Tournaments tab and Club profile Past Results tab also reference the tournament detail page
✅Search — global, no dedicated search page
Global search covers all content: players, clubs, tournaments, groups, listings. No separate search results page at this stage — the platform is small enough that a global dropdown/overlay handles it.
Search covers: players, clubs, games/tournaments, groups, marketplace listings
Results surface inline — no /search route needed at v1 scale
✅Club admin — contextual + Club Hub tab
No separate /admin dashboard. Admin actions appear inline wherever the content lives, plus a lightweight Club Hub tab for cross-page tasks.
Contextual: Edit button on club profile, tournament pages, event pages when logged in as admin
Club Hub tab (on club profile): broadcast messaging, all active/upcoming tournaments, score disputes, admin-only scoring lock
✅Groups — My Groups / Global Groups / Starred
Groups nav section has three views. My Groups is the default. Global Groups is the discovery directory. Starred pins the most-used groups to the top.
My Groups tab: all joined/created groups, starred at top
Global Groups tab: directory of all open and filtered groups, filterable
Club-managed groups also discoverable from club profiles and Global Groups directory
✅Marketplace — single landing, hashtag filtering
One Marketplace page for all listings. Equipment and coaching listings live in the same feed. Creators tag listings with hashtags at creation; the feed is filterable by any hashtag combination.
No separate Equipment / Coaching pages
Coaching listings pull from player profile (tier + match history as credibility signal)
Zero transactions — connection layer only (Facebook Marketplace model)
